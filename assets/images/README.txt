# PHOTO PLACEHOLDER

Place your profile photo here as `profile.jpg`.

To display it in the portfolio, open `index.html` and in the `#hero-avatar` div, replace:

    <svg class="avatar-svg" ...>...</svg>

with:

    <img src="assets/images/profile.jpg" alt="Saroj Kumar – Full Stack Web Developer" class="avatar-img" />

The CSS is already set up to display a round, full-size photo.
